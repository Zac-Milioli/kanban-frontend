# Kanban Frontend

## NOTA
Para o adequado funcionamento, o backend também deve estar sendo executado.
> https://www.github.com/zac-milioli/kanban-api-backend

## Setup / Instalação
```
npm install
```

## Compilar e executar
```
npm run serve
```
> O projeto estará disponível em http://localhost:8080
